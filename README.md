# gdrive-client

Google Drive Client


## 設定

### Google Drive APIの利用登録

[Google Drive Console](https://console.developers.google.com/) で、
認証情報(JSONファイル)を作成し、ダウンロードする。

1. プロジェクトを作成
2. 「Google Drive API」を`有効にする`
3. 認証情報(`OAuth クライアント ID`)を作成
4. 認証情報(JSONファイル)をダウンロード
